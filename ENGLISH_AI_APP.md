# English AI Learning App

هذا الفرع مخصص لتطبيق تعلم اللغة الإنجليزية بالذكاء الاصطناعي، وهو منفصل عن فرع `main` حتى لا يتأثر مشروع غرفة الطوارئ.

## المراحل المنجزة

### المرحلة الأولى

- التسجيل وتسجيل الدخول عبر Supabase Auth.
- إعداد ملف المتعلم والهدف واللهجة والاهتمامات والوقت اليومي.
- NestJS API مرتبط بـ PostgreSQL/Supabase.

### المرحلة الثانية

- اختبار تحديد مستوى تكيفي CEFR من A1 إلى C2.
- تقييم مستقل للقواعد والمفردات والقراءة والاستماع.
- تخزين كل جلسة وإجابة ونتيجة داخل قاعدة البيانات.

### المرحلة الثالثة

- صفحة رئيسية ذكية ومستوى مستقل لكل مهارة.
- هدف يومي وXP وسلسلة تعلم ومراجعات مستحقة.
- مسار أسبوعي شخصي من 7 إلى 14 نشاطاً.
- خوارزمية تراعي نقاط الضعف والهدف والاهتمامات.

### المرحلة الرابعة

- محرك دروس تفاعلي قابل للحفظ والاستئناف.
- الاستماع والقراءة والمفردات والأسئلة والتصحيح من الخادم.
- حفظ المحاولات والدرجات وإكمال المسار تلقائياً.

### المرحلة الخامسة

- مختبر تحدث بثلاثة أوضاع: القراءة الموجهة، Shadowing، وRole-play.
- تسجيل WAV من Flutter ورفع آمن بحد 15 MB.
- تفريغ اختياري عبر OpenAI وتقييم نطق اختياري عبر Azure.
- درجات الدقة والطلاقة والاكتمال والتنغيم والكلمات والفونيمات.
- محادثة صوتية متعددة الأدوار مع بديل محلي آمن للتطوير.
- تخزين التسجيلات في حاوية Supabase Storage خاصة، مع بديل محلي للتطوير.
- تسجيل وإنشاء حساب وتأكيد البريد وإعادة تعيين كلمة المرور.
- أدوار `learner` و`admin` في قاعدة البيانات مع `AdminGuard`.
- لوحة إدارة لمؤشرات المستخدمين والدروس والنشاط ومحاولات التحدث.
- سكربت آمن لإنشاء أو ترقية الحساب الإداري دون تخزين كلمة المرور في المصدر.
- GitHub Actions لفحص Backend وFlutter، وتطبيق migrations، وإنشاء المدير يدوياً من Secrets.
- Supabase migration رقم `202606260005_speaking_and_admin.sql`.

## الحساب الإداري

بيانات الدخول لا تُكتب في هذا المستودع. بعد إضافة الأسرار التالية يمكن تشغيل workflow باسم `Bootstrap Administrator`:

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `DATABASE_URL`
- `ADMIN_EMAIL`
- `ADMIN_PASSWORD`

ويظل الحساب الناتج مستخدماً تعليمياً عادياً بالإضافة إلى صلاحية الإدارة.

## المراحل الأساسية المتبقية للإصدار الأول

تبقى أربع مراحل رئيسية:

1. الكتابة الذكية والتصحيح النحوي والأسلوبي.
2. المفردات والإملاء ونظام FSRS.
3. لوحة صناعة المحتوى والإشعارات والتحليلات المتقدمة.
4. الاختبارات الشاملة والأمان والنشر الإنتاجي ومتاجر التطبيقات.

لا ترفع كلمة مرور قاعدة البيانات أو Service Role Key أو مفاتيح الذكاء الاصطناعي أو بيانات المدير داخل الملفات العامة.
