# English AI Learning App

هذا الفرع مخصص لتطبيق تعلم اللغة الإنجليزية بالذكاء الاصطناعي، وهو منفصل عن فرع `main` حتى لا يتأثر مشروع غرفة الطوارئ.

## المراحل المنجزة

1. التسجيل وتسجيل الدخول وإعداد ملف المتعلم عبر Supabase Auth.
2. اختبار تحديد مستوى تكيفي CEFR من A1 إلى C2.
3. صفحة رئيسية ذكية ومسار أسبوعي شخصي وXP وسلسلة تعلم.
4. محرك دروس تفاعلي للاستماع والقراءة والمفردات والتمارين.
5. مختبر تحدث: Read Aloud وShadowing وRole-play، مع أدوار إدارية ولوحة مراقبة.
6. الكتابة الذكية والتصحيح النحوي والأسلوبي.

## المرحلة السادسة: الكتابة الذكية

- مهام كتابة متدرجة من A1 إلى C2: جمل، فقرات، رسائل، قصص، مقالات، وتقارير.
- تقييم سبعة محاور: الدرجة العامة، القواعد، الإملاء، المفردات، الترابط، إنجاز المهمة، والأسلوب.
- نص مصحح يحافظ على المعنى ونسخة محسنة أقوى أسلوبياً.
- ملاحظات موضعية مع شرح عربي واقتراحات مفردات أقوى وأمثلة.
- حفظ المسودة وكل المراجعات ومقارنة التحسن بين الإصدارات.
- Structured Outputs عبر OpenAI Responses API مع محرك محلي احتياطي.
- حماية من تعليمات Prompt Injection داخل نص المتعلم.
- حساب استخدام الذكاء الاصطناعي دون إظهار سجلات المزود للعميل.
- إضافة XP وتحديث السلسلة بعد التقييم.
- Migration: `202606260006_intelligent_writing.sql`.
